## Campus Club Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/5c8c0f83592e48f7b9abeee64334a25e/ghb20250919) 


### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `SpringBoot` / `MyBatis` / `JDK1.8`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` / `Element-UI`, etc.
- Database: `MySQL8.0` / `MyBatis`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- `Frontend Environment Node14`
- etc.

### 3. Feature Description
The system is divided into (roles): club members, administrators.

**System Features:**
1. Login
2. System Homepage
3. System User Management
4. Club Type Management
5. Club Information Management
6. Club Member Management
7. Club Activity Management
8. Notification Information Management
9. Club Application Records
10. Fee Record Management
11. Registration
12. Secure Logout
13. Permission Differentiation
